<h3>O'rnatish:</h3>
  
```
git clone https://github.com/UsmonHamidulloh/i3wm-dotfiles
cd i3wm-dotfiles/
cp -r alacritty/ ~/.config/
cp -r i3/ ~/.config/
cp -r polybar/ ~/.config/   
cp -r wallpaper/ ~/.config/
cp picom.conf ~/.config/
```
